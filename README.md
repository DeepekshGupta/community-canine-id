# Community Canine ID (CCID)
CCID is an open-source, serverless platform designed to track the medical history, location, and wellbeing of street dogs (specifically targeted at the Indian context). By utilizing passive QR tags attached to collars, the project decentralizes the cost of monitoring from a central server to the user's smartphone. It supports both offline-first static data and online-first dynamic records to ensure resilience in areas with poor connectivity.

By leveraging QR code technology, caregivers, NGOs, and the general public can access and update a dog’s health records, ensuring continuity of care even when individual caretakers are unavailable.
